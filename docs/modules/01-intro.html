What Is Kharcha Track?

Your expense tracker — and the code behind it

Kharcha Track is a personal expense tracker with superpowers. "Kharcha" means "expense" in Nepali. It does what any good tracker does — records what you spend — but it also predicts your future spending and detects unusual purchases automatically.

Think of it as a financial diary that reads itself back to you and warns you about unusual pages. By the end of this course, you will understand every line of code that makes it work.

The Technology Behind It

Four main technologies work together to make Kharcha Track run. Each handles a different job.

⚙️
Laravel 12

The engine that powers everything — handles requests, talks to the database, enforces rules.

⚡
Livewire 4

Makes pages interactive without writing JavaScript separately — like magic HTML that updates itself.

🎨
MaryUI + Tailwind CSS

The visual layer — beautiful buttons, tables, and forms out of the box.

🗄️
PostgreSQL

The filing cabinet — stores every expense, category, and user permanently.

All of these are built on PHP, a server-side language, and follow a framework pattern that keeps code organized.

protected static function booted(): void
{
    static::created(function ($expense) {
        $expense->histories()->create([
            'user_id' => $expense->user_id,
            'title' => $expense->title,
            'amount' => $expense->amount,
            'action' => 'create',
        ]);
    });
}

Key Insight

This pattern — "when X happens, automatically do Y" — is called an event listener. It is everywhere: when you get an email, your phone dings. Kharcha Track uses it so every expense change is automatically journaled.
