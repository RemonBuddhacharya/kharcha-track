# KharchaTrack: Smart Expenses Tracker with Insights

Welcome to the KharchaTrack tutorial series! This comprehensive guide will help you build a complete expense management and forecasting application from scratch.

## 🌟 Project Overview

KharchaTrack is a sophisticated expense management system that leverages machine learning algorithms to provide intelligent insights, forecast future expenses, and detect unusual spending patterns. This tutorial series will guide you through building the entire application step by step.

## 📚 Tutorial Series

Follow these tutorials in sequence to build the complete KharchaTrack application:

1. [Getting Started with KharchaTrack](tutorials/01-getting-started.md) - Project setup and environment configuration
2. [Database Setup](tutorials/02-database-setup.md) - Creating migrations, models, and seeders
3. [Authentication & User Management](tutorials/03-authentication.md) - Implementing user authentication and roles
4. [Frontend Development](tutorials/04-frontend.md) - Setting up Mary UI and frontend components
5. [Expense Management](tutorials/05-expense-management.md) - Creating the expense tracking functionality
6. [Forecasting System](tutorials/06-forecasting.md) - Building the expense forecasting feature
7. [Anomaly Detection](tutorials/07-anomaly-detection.md) - Implementing the anomaly detection system

## 🎯 What You'll Learn

By completing this tutorial series, you'll learn how to:

- Create a full-stack Laravel application with authentication
- Implement Livewire for interactive components
- Use Mary UI for beautiful, responsive interfaces
- Work with PostgreSQL for data storage
- Implement machine learning algorithms in PHP
- Build data visualization with charts
- Create a role-based permission system
- Develop a comprehensive dashboard
- Implement CRUD operations with real-time updates
- Add forecasting capabilities to predict future expenses
- Create an anomaly detection system to identify unusual spending

## 🔧 Technical Stack

- **Framework**: Laravel 11
- **Database**: PostgreSQL 16
- **Machine Learning**: PHP-ML
- **Frontend**: Mary UI
- **Authentication**: Laravel Breeze
- **Roles & Permissions**: Spatie Laravel Permission
- **Real-time Components**: Livewire

## 📋 Project Structure

The project code is organized into several key areas:

- **Models**: User, Expense, ExpenseHistory
- **Livewire Components**: ExpenseForm, ExpenseList, ExpenseDashboard, ExpenseForecasting, AnomalyDetection
- **Services**: ForecastingService, AnomalyDetectionService
- **Frontend Views**: Blade templates with Mary UI components
- **Database**: Migrations, seeders, and schema

## 🚀 Prerequisites

Before starting the tutorials, make sure you have the following installed:

- PHP 8.2+
- Composer
- Node.js and npm
- PostgreSQL
- Git
